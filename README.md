# FridgeMatch

Progetto per Tencologie e Sistemi Web


Principali funzionalità:

* lista della spesa
* frigo/dispensa virtuale
* scadenziario intelligente
* community ricette

SETUP DEL PROGETTO

Dopo aver clonato la repository, è necessario installare le dipendenze del backend

Passaggi:
1. Entrare nella cartella del server:
    cd server
2. Installare le dipendenze:
    npm install
3. Avviare il server:
    npm start
